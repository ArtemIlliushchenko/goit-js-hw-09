# Домашнє завдання 9 – SimpleLightbox + Форма зворотного зв'язку з localStorage

## Опис
Дев’яте домашнє завдання з курсу JavaScript.  
Практика рефакторингу галереї з використанням сучасної бібліотеки SimpleLightbox (npm), а також робота з формами, локальним сховищем, делегуванням подій input та валідацією перед відправкою.

## Технології / стек
- JavaScript (ES6+)
- Модульний підхід (import / export)
- Пакетний менеджер: **npm**
- Бібліотека: **SimpleLightbox** (npm-пакет)
- Локальне сховище: `localStorage`
- Події: `input`, `submit`
- Делегування подій на формі
- Робота з формами (`elements`, `value`, `reset`)
- Шаблонні рядки / динамічна розмітка
- CSS (стилізація за макетом)

## Функціонал

| Файл              | Завдання                              | Що робить                                                                                   |
|-------------------|---------------------------------------|---------------------------------------------------------------------------------------------|
| 1-gallery.html    | Структура галереї                     | Містить `<ul class="gallery"></ul>`                                                         |
| 1-gallery.js      | Галерея з SimpleLightbox              | Динамічно створює картки, ініціалізує SimpleLightbox з captions (alt) через 250 мс         |
| 1-gallery.js      | Рефакторинг з ДЗ8                     | Видалено делегування та basicLightbox, додано npm-залежність та правильний імпорт стилів   |
| 2-form.html       | Форма зворотного зв’язку              | Містить поля email, message та кнопку submit                                                |
| 2-form.js         | Збереження стану форми                | Зберігає дані в localStorage під ключем "feedback-form-state" при кожній зміні input       |
| 2-form.js         | Відновлення даних при завантаженні    | Заповнює форму даними з localStorage при завантаженні сторінки                              |
| 2-form.js         | Валідація та сабміт                   | Перевіряє заповненість, виводить об’єкт у консоль, очищає форму + сховище + formData       |

## Структура проєкту (основні команди)

```bash
# Встановлення залежностей
npm install

# Встановлення SimpleLightbox (якщо ще не встановлено)
npm install simplelightbox
```
## Приклади роботи
Завдання 1 – галерея

9 карток з preview-зображеннями
Клік → відкривається модальне вікно SimpleLightbox
Через 250 мс знизу з’являється підпис (вміст атрибута alt)
Працює гортання стрілками, закриття Esc / кліком поза зображенням

Завдання 2 – форма зворотного зв’язку
JavaScript// Приклад об’єкта в консолі після успішного сабміту
{
  email: "user@example.com",
  message: "Добрий день! Хочу замовити консультацію."
}

// Якщо не всі поля заповнені:
alert("Fill please all fields")

Дані зберігаються при кожній зміні
При перезавантаженні сторінки поля заповнюються автоматично
Після успішного сабміту — все очищається (форма, localStorage, formData)
